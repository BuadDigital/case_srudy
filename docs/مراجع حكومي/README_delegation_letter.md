## المنطق المعتمد (ملخص حاكم)

- الخطاب موجّه للدائرة. مفتاح التجميع = **المحكمة + الدائرة**.
- كل صك يحمل أمر عمله ودائرته → الخطاب يجمع عقارات بأوامر عمل متعددة.
- جدول الخطاب: **م | أمر العمل | رقم الصك | المالك | رقم الطلب**.
- الأعمدة الرقمية مضبوطة على المحتوى، عمود المالك مرن (يمتصّ الباقي ويلتفّ للأسماء الطويلة).
- الرقم المرجعي بالترميز الموحّد، يُثبَّت مع snapshot لحظة الإصدار.
- الطباعة على الترويسة الرسمية (خلفية بيضاء + صورة A4 تملأ الصفحة).

التفاصيل الكاملة في الملفات أعلاه.
