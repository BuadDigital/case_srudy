# مصادر بيانات الرفع لإنفاذ — مَن يوفّر كل حقل وفي أي شاشة

> الغرض: تحديد **مصدر كل حقل** (الدور صاحب «أصيل» الذي يُدخله، ومَن «يعتمده» إن وُجد) في نظام إجادة، حتى يبني Cursor كل حقل ضمن نافذة المستخدم الصحيح. طبقة واحدة لمصدر الحقيقة تغذّي تبويب «بيانات الرفع لإنفاذ».

## الأدوار/الشاشات (المصادر)

| الدور | الشاشة | يوفّر |
|---|---|---|
| **معاين** | شاشة المعاين | المشاهدات الميدانية للموقع والمبنى |
| **هندسي** | شاشة المكتب الهندسي | الرفع المساحي والحدود والأطوال |
| **مقيّم** | شاشة المقيّم (المقياس) | القيم والمنهجية وبيانات التقييم |
| **مراجع** | شاشة المراجع الحكومي | المحاكم والوقف والمفاتيح |
| **بورصة** | البورصة العقارية | بيانات الصك (مرجعية) |
| **نظام** | الحسابات / البيانات المرجعية للمعاملة | يأتي تلقائياً |
| **أخصائي** | شاشة أخصائي دراسة الحالة | الاعتماد وملاحظات |
| **مالك/عميل** | وثائق المالك أو أطراف التنفيذ | مستندات يقدّمها العميل |

**ملاحظة على «يعتمدها»:** بعض الحقول مصدرها (أصيل) دورٌ، ويعتمدها دورٌ آخر قبل الرفع (نموذج: أصيل = المعاين، يعتمد = المقيّم).
**حالة المصدر:** ✅ مؤكّد من المستخدم · ⚠️ مُستنتَج (يحتاج تأكيد).

## يتبقّى تأكيدك على الحقول الموسومة ⚠️

1. **بيانات التقرير:** عنوان المقيم، رقم تواصل المقيم، تاريخ إصدار التقرير — وضعتها «النظام (حساب المقيّم)»؛ هل هي كذلك أم من شاشة المقيّم؟
2. **الحدود الأربعة + أطوالها:** وضعتها «المكتب الهندسي»؛ تأكيد؟
3. **الأصول المرتبطة (هل مرتبط؟ + ملاحظات الربط):** وضعتها «الأخصائي»؛ تأكيد؟
4. **استلام المفاتيح (السؤال + الإثبات):** وضعتها «المراجع الحكومي»؛ تأكيد؟
5. **ملاحظات «معلومات أخرى» و«البيانات الختامية»:** وضعتها «الأخصائي»؛ تأكيد؟
