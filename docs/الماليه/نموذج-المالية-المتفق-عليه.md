# نموذج المالية المتفق عليه

> مرجع الاتفاق التشغيلي. للتفاصيل المطبّقة حالياً راجع [دورة موظف المالية](../finance-officer-workflow.md) و[مسار أتعاب الأطراف](../inspector-fees-billing-workflow.md).

## المخطط الذهني

```mermaid
mindmap
  root((المالية))
    الدخل من إنفاذ
      دخل دراسة المعاملة
      دخل تكاليف الرفع
    المصروفات
      مكتب هندسي
        نظير الرفع المساحي
      متعاونون
        المعاين
          فرد — تسعيرة
          شركة — تسعيرة مختلفة
        المراجع الحكومي
```

## 1) الدخل (وارد من إنفاذ)

| البند | الوصف |
|--------|--------|
| دخل على إجمالي دراسة المعاملة | `CaseStudyFeeSar` |
| دخل لتكاليف الرفع | `SurveyFeeSar` |

**المطبّق:** بندَان لكل عقار + مجموع للفاتورة وVAT 15٪. الحفظ يقبل أيضاً المبلغ القديم القديم للتوافق.

## 2) المصروفات (صادر)

| البند | التسعيرة الافتراضية |
|--------|---------------------|
| مكتب هندسي (متعاون شركة / موظف) | 500 / 150 |
| المعاين — متعاون فرد | 400 |
| المعاين — متعاون شركة | 500 |
| المعاين — موظف | 100 |
| المراجع الحكومي — فرد / شركة / موظف | 350 / 450 / 100 |

**المطبّق:** سجلات أتعاب عند التوزيع لـ `field-inspection` و `engineering-survey` و `government-review`. التسمية القديمة «متعاون» تُعامل كفرد (معاينة) أو كمكتب خارجي (رفع).

## 3) مسار الصرف المطبق (أتعاب المعاينة والرفع والمراجعة)

```
مسودة → بانتظار المشرف → جاهز لدى المالية → أمر صرف → مصروف
         ↕ إرجاع / استفسار
```
